Welcome to test Tensor Flow!

This is the front-end for testing TensorFlow based image recognition engine available at {{serverUrl}}

The enginge is built by slightly modifying the Google's MNIST example to process also user provided images. There is also a simple image processing to modify JPG images to match the 28x28 format that the MNIST sample uses. Then finally the whole thing has been cloudified by putting it on the Heroku server.
